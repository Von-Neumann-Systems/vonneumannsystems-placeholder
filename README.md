# Von Neumann Systems LLC

Static website for [vonneumannsystems.com](https://vonneumannsystems.com), hosted with GitHub Pages.

## Structure

- `index.html` — complete single-page site, including styles
- `assets/award-vault-logo.png` — featured product artwork
- `CNAME` — custom-domain configuration for GitHub Pages

There is no build step and there are no third-party dependencies. To preview locally, open `index.html` in a browser or serve the repository with any static HTTP server.
